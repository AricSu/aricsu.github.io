TIHC 是一个面向 TiDB troubleshooting 的 Chat 系统：

- `tihc/`：浏览器扩展（sidepanel UI）
- `tihc-serverless/`：Vercel serverless 后端（OpenAI / Manus / TiDB doc GraphRAG）
- `evalite-demo/`：评估工程（问题空间收敛 + 自评校准）

核心原则：

- 默认不存储用户对话/内容（stateless）
- 必须存时，优先使用 TiDB Serverless（并只存最小必要字段、可加 TTL）

下一步建议从 Quickstart 开始：`/tihc/quickstart`
