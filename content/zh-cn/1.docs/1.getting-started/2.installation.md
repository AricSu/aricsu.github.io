## 快速开始

你可以通过脚本一键安装（macOS）

```bash [Terminal]
curl -fsSL https://raw.githubusercontent.com/AricSu/tihc/main/scripts/install.sh | bash
```

也可以从 GitHub 发布页下载预编译的二进制文件：：

 - 访问 TiHC Releases 页面
 - 下载对应操作系统的二进制文件
 - 解压下载的文件
```bash [Terminal]
tar -xvf tihc-darwin-amd64.tar.gz
chmod +x tihc

# 查看帮助信息
./tihc --help

# 输出
tihc 1.0.0
Author: Aric
TiHC CLI Tool

Doc: https://www.askaric.com/en/tihc

USAGE:
    tihc [OPTIONS] <SUBCOMMAND>

    -l, --log-file <LOG_FILE>      Log file path [default: tihc_started_at_20250320_142358.log]
    -L, --log-level <LOG_LEVEL>    Log level (trace, debug, info, warn, error) [default: info]
    -V, --version                  Print version information

SUBCOMMANDS:
    collect    Collect info from TiDB components (tidb, tikv, pd, br)
    help       Print this message or the help of the given subcommand(s)
```

## 源码构建

从 GitHub 下载源码：

1. 打开 <https://github.com/nuxt-ui-templates/saas>
2. 点击 `Use this template` 按钮
3. 输入仓库名并点击 `Create repository from template`
4. 克隆你的新仓库
5. 用你喜欢的包管理器安装依赖
6. 启动开发服务器

就是这样！你现在可以在 [`content/`](https://content.nuxt.com/usage/content-directory) 目录下编写文档了 🚀


```
# 克隆代码库
git clone https://github.com/AricSu/tihc.git

# 编译
cd tihc
make

# 运行
./tihc --help
```
