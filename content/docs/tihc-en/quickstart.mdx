## 1) Deploy `tihc-serverless`

Set these env vars in Vercel:

- `OPENAI_API_KEY`
- `GOOGLE_CLIENT_ID`
- `GOOGLE_WORKSPACE_DOMAIN` (e.g. `pingcap.com`)

Optional:

- `TIDB_API_URL`, `TIDB_API_TOKEN` (enable `tidb` / `openai_rag`)
- `MANUS_API_URL`, `MANUS_API_TOKEN` (enable `manus`)

## 2) Run `tihc` extension

- `cd tihc && npm i && npm run dev`
- Open the sidepanel
- Top settings:
  - `Backend=serverless`
  - `API=https://YOUR.vercel.app`
  - `Engine=openai` (or `openai_rag` / `manus` / `tidb`)
  - `Google Client ID=<client id>` â†’ click `Login`

If serverless returns 401, the token likely fails Workspace domain or client id validation.
