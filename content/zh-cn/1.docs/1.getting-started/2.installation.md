## 快速开始

你可以通过以下命令新建项目：

```bash [Terminal]
npx nuxi init -t github:nuxt-ui-templates/saas
```

或从 GitHub 创建新仓库：

1. 打开 <https://github.com/nuxt-ui-templates/saas>
2. 点击 `Use this template` 按钮
3. 输入仓库名并点击 `Create repository from template`
4. 克隆你的新仓库
5. 用你喜欢的包管理器安装依赖
6. 启动开发服务器

就是这样！你现在可以在 [`content/`](https://content.nuxt.com/usage/content-directory) 目录下编写文档了 🚀
