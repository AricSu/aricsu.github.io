
# TiHC Collect
TiHC Collect 是一个用于采集 TiDB 组件的 Profile 的工具。它可以采集 TiDB 组件的 Profile，并将其保存到本地文件中。
## 安装
```bash
go install github.com/aricSu/TiHC-Collect@latest
```
## 使用
```bash
tihc-collect --config config.yaml
```

| --- | --- |
