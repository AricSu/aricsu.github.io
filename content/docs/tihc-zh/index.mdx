TiHC 是一个面向 TiDB 排障的浏览器侧边栏助手。它把 TiDB Docs 做成 GraphRAG 索引，让你在当前页面旁边直接提问、获得流式回答，并把最近的对话保存在本地（默认不落库）。

## 快速开始

1. 安装扩展：在 Chrome Web Store 点击「添加至 Chrome」。
2. 打开侧边栏：在浏览器工具栏里打开 TiHC Side Panel。
3. 提问：把遇到的现象、日志片段、版本信息贴进去（越具体越好）。

示例问题：

```text
TiKV 99% latency 突然变高，应该从哪些指标/日志开始排查？
```
