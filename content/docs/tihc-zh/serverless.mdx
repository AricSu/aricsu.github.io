## Endpoint

`POST /api/stream_chat`

Request JSON:

```json
{ "messages": [{ "role": "user", "content": "..." }], "chat_engine": "openai", "stream": true }
```

Response:

- streaming text（`text/plain; charset=utf-8`）

## Engines

- `openai`
- `openai_rag`（GraphRAG → OpenAI）
- `manus`（proxy）
- `tidb`（proxy）
