本模板是一个开箱即用的 SaaS 解决方案，基于 [Nuxt UI](https://ui.nuxt.com) 快速创建美观且响应式的 Nuxt 应用。

模板包含可定制的落地页、价格页、文档、博客和认证页面（登录、注册）。

## 主要特性

本模板包含一系列功能，助力文档管理：

- **基于** [**Nuxt**](https://nuxt.com)：采用最新 Nuxt 框架，性能优异。
- **构建于** [**Nuxt UI**](https://ui.nuxt.com)：集成丰富的 UI 组件。
- [**MDC 语法**](https://content.nuxt.com/usage/markdown) **通过** [**Nuxt Content**](https://content.nuxt.com)：支持 Markdown 与组件集成，实现动态内容。
- **自动生成侧边栏导航**：根据内容结构自动生成导航。
- **全文搜索**：内置搜索功能，便于内容检索。
- **优化排版**：精致排版，提升可读性。
- **深色模式**：支持深色模式，满足用户偏好。
- **丰富功能**：探索模板，体验更多能力。
