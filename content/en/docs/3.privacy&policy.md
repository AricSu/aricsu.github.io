# Privacy Policy

**Last updated:** December 2025

## 1. Introduction

TiHC (“we”, “our”, or “the extension”) is a Chrome extension designed for TiDB users. It provides an AI-powered assistant in the browser sidebar to support interactive Q&A, technical suggestions, and code-related help.

We take user privacy seriously. This Privacy Policy explains what data TiHC collects, how it is used, and how it is protected.

---

## 2. Data We Collect

TiHC **does not collect any personally identifiable information (PII)**.

Depending on how you use the extension, the following data may be processed:

### 2.1 User Input

* Questions or messages you type into the chat interface
* These inputs are used **only to generate AI responses**

### 2.2 Local Chat History

* Up to **20 recent messages**
* Stored **locally in your browser only**
* Never uploaded or shared automatically

### 2.3 Technical Metadata (Non-Personal)

* Browser-generated technical data required for extension functionality (e.g. runtime state)
* No tracking identifiers, cookies, or fingerprinting mechanisms are used

---

## 3. How We Use Data

Collected data is used **solely to provide core functionality**, including:

* Generating AI-powered responses
* Displaying streaming answers
* Rendering Markdown content
* Enabling copy-to-clipboard for code snippets

We **do not**:

* Sell user data
* Use data for advertising or analytics
* Track user behavior across websites

---

## 4. Data Storage & Retention

* Chat history is stored **locally on your device**
* Maximum of **20 messages**, automatically overwritten
* You can clear the data at any time by removing the extension or clearing browser storage

No server-side long-term storage is performed by the extension itself.

---

## 5. Data Sharing

TiHC **does not share user data with third parties**, except where strictly required to process AI responses (e.g. sending user input to an AI service endpoint).

Such processing:

* Is limited to the request context
* Does not include user identity
* Is not used for model training by default

---

## 6. Permissions Justification

TiHC requests only the minimum permissions required to function:

* **Storage** – to save local chat history
* **Active tab / side panel access** – to display the assistant UI

No access to browsing history, page content, or personal accounts is performed unless explicitly triggered by the user.

---

## 7. Security

We apply reasonable technical measures to protect user data, including:

* Local-only storage for chat history
* No embedded trackers or analytics SDKs
* Minimal permission scope

---

## 8. Children’s Privacy

TiHC is **not intended for children under 13**. We do not knowingly collect data from children.

---

## 9. Changes to This Policy

This Privacy Policy may be updated from time to time. Any changes will be reflected in the extension listing or accompanying documentation.
